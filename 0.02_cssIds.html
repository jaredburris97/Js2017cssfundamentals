CSS id's

An ID is a UNIQUE identifier

Id notes

  • Is unique to the element that it's on
  • to target it from our css we need to use a # sign
  • you can use them with classes on the same element
  • Should only be used ONCE per page - they are unique! Important with jQuery later on
  • Use for one off styling

Ids versus Classes

ids

This id will be linked in the CSS page by a #

You can't have two ids on an element.

classes

one class here.

Two classes here.

You can use an id twice, but it's best not to. It should be used mostly for targeting specific items.
